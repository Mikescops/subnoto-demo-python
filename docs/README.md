# Subnoto SDK demo (Python) – Documentation

This repo is a demo app that shows how to use the Subnoto API from Python and embed the signing flow in an iframe. It includes:

- **Quote**: Build a quote PDF from a form (ReportLab), create an envelope with Smart Anchors (recipient and signature block are read from the PDF), send with no email, and open the signing experience in an iframe.

Use it as a reference when integrating the Subnoto API and embedding the signing flow in your own product.

## Docs

- [Getting started](getting-started.md): Prerequisites, setup, env vars, and how to run the demo.
- [Customer implementation](customer-implementation.md): Step-by-step tutorial to implement the same logic on your side (API client, envelope creation with Smart Anchors, iframe token).
